Project Overview

Chatbot รับออเดอร์และบันทึกคำสั่งซื้ออัตโนมัติ

กรณีศึกษาจำลองสำหรับ chatbot รับออเดอร์ผ่านแชต ที่ช่วยสรุปรายการ ยืนยันคำสั่งซื้อ รับสลิปโอนเงิน และบันทึกข้อมูลคำสั่งซื้อเข้าสู่ workflow ต่อแบบอัตโนมัติ โดยใช้ข้อความและหน้าจอจำลองแทนข้อมูลจริง

ปัญหา

รับออเดอร์ทางแชตแล้วต้องมาสรุปรายการเอง ตรวจยอดเอง ไล่สลิปเอง และคัดลอกข้อมูลลงระบบอีกหลายรอบ

แนวทาง

ให้ chatbot รับออเดอร์ สรุปรายการ ยืนยันยอด รับสลิป และส่งข้อมูลต่อเข้า workflow หรือฐานข้อมูลคำสั่งซื้ออัตโนมัติ

ผลลัพธ์

ลดงานแอดมิน ลดการตกหล่นของออเดอร์ และทำให้ข้อมูลพร้อมส่งต่อไปจัดส่งหรือกระทบยอดได้เร็วขึ้น

Sales Flow Showcase

LINE OA / Order Flow

LINE Chatbot ที่ตอบคำถาม รับออเดอร์ และดึงข้อมูลเข้าระบบได้อัตโนมัติ

เหมาะกับงานที่ต้องให้ลูกค้าคุยผ่าน LINE แล้วระบบช่วยตอบคำถาม สรุปรายการ ยืนยันยอด รับสลิป และส่งข้อมูลต่อเข้า spreadsheet, database หรือ workflow หลังบ้านโดยอัตโนมัติ

1. ลูกค้าสั่งสินค้าในแชตแบบเดิม โดยไม่ต้องเรียนรู้ระบบใหม่
2. bot สรุปออเดอร์และยอดรวมให้ยืนยันได้ทันทีในหน้าคุยเดียว
3. ทีมงานได้ข้อมูลพร้อมใช้งานต่อ ทั้งสลิป ยอด และรายการสั่งซื้อ

Before

ทีมต้องตอบแชตเอง สรุปรายการเอง เช็กยอดเอง และคีย์คำสั่งซื้อซ้ำเข้าระบบ ทำให้เสียเวลาและตกหล่นง่าย

After

ลูกค้าสั่งงานตาม flow เดียวกันทุกครั้ง ระบบได้ข้อมูลออเดอร์พร้อมยอดและหลักฐานโอน ทีมจึงโฟกัสเฉพาะการตรวจสอบและส่งต่อ

Confidentiality Note

ผลงานชิ้นนี้เป็น mockup ที่อ้างอิงจาก use case การทำงานจริง โดยมีการปรับข้อความ บทสนทนา ชื่อระบบ และรายละเอียดบางส่วนเพื่อรักษาความลับของลูกค้า
