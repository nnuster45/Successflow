Project Overview

ระบบจัดการเอกสารบัญชีอัตโนมัติ

ออกแบบ workflow automation สำหรับงานสรุปยอดขายและส่งบัญชีรายสัปดาห์ ที่ต้องดึงข้อมูลจากหลายแพลตฟอร์ม e-commerce, ระบบภายใน, ขนส่ง, ยอดเก็บปลายทาง, slip โอนเงิน และเอกสารอื่น ๆ เพื่อจัดทำใบกำกับภาษี, ยอด commission และรายการส่งบัญชีได้อย่างเป็นระบบ

ปัญหา

ทุกสัปดาห์ทีมต้องรวมยอดขาย, commission, โปรโมชั่น, ค่าหัก, ขนส่ง, COD และหลักฐานรับชำระจากหลายแหล่งเพื่อส่งบัญชี

แนวทาง

วาง workflow สำหรับรับไฟล์, OCR slip, map ข้อมูลจาก e-commerce, ERP, ขนส่ง และระบบภายใน ก่อนจัดทำ output สำหรับบัญชี

ผลลัพธ์

ลดเวลาปิดรอบงานรายสัปดาห์ ลดการเทรนงานซ้ำ และทำให้ความรู้กระบวนการไม่ผูกกับคนใดคนหนึ่ง

Project Story

สถานการณ์ตั้งต้นคือทุกสัปดาห์ทีมขายและทีมบัญชีต้องสรุปรายการเพื่อส่งบัญชี ออกใบกำกับภาษี และคำนวณ commission ให้พนักงานขายแต่ละคนตามเงื่อนไขการจ่าย โดยต้องดึงข้อมูลจากหลายแพลตฟอร์ม e-commerce, ระบบภายใน, ขนส่ง, ยอดเก็บปลายทาง, โปรโมชั่น, ค่าหัก และ slip โอนเงินจากลูกค้า บางส่วนมาในรูปภาพหรือ PDF ทำให้ต้อง reconcile ด้วยมือ ใช้เวลาเทรนงานสูง และเสี่ยงสะดุดทันทีเมื่อคนรับผิดชอบเปลี่ยน

Process Demonstration

วางภาพจำลองหรือ flow ของงานตรงนี้ได้ เช่น ดึงยอดจากหลายแพลตฟอร์ม, อ่าน OCR slip, map commission และค่าหัก, reconcile ยอด และเตรียมไฟล์สำหรับส่งบัญชีโดยไม่เปิดข้อมูลลูกค้าจริง

Before / After

Before

เปิดหลายแพลตฟอร์ม, export หลายไฟล์, อ่านสลิป, รวมยอดขาย, commission, โปรโมชั่น, ค่าหัก, COD และเช็กยอดด้วยมือ ใช้เวลาหลายชั่วโมงถึงหลายวันต่อรอบ และต้องพึ่งคนที่รู้ flow เดิมอยู่แล้ว

After

ผู้ใช้กดรัน workflow หนึ่งครั้ง ระบบดึงข้อมูลจากหลายแหล่ง อ่าน OCR slip, จับคู่ข้อมูลตาม rule, คำนวณรายการที่เกี่ยวข้อง และเตรียม output สำหรับส่งบัญชีหรือให้ทีมตรวจต่อได้ทันที

How It Works

01

รับข้อมูลจากแพลตฟอร์มขาย, ระบบภายใน, ขนส่ง, COD, statement หรือหลักฐานการโอนเข้ากระบวนการเดียว

02

OCR และ normalize ข้อมูลจาก slip, PDF หรือไฟล์ต้นทางให้อยู่ใน format ที่พร้อมใช้ต่อ

03

ระบบ map ยอดขาย, commission, ค่าหัก, โปรโมชั่น และรายการชำระตาม rule พร้อม flag exception ที่ต้องตรวจสอบ

04

ทีมรับเฉพาะรายการที่ต้องตัดสินใจต่อ พร้อมใช้ output สำหรับส่งบัญชี ออกเอกสาร หรือสรุป commission ได้ทันที
