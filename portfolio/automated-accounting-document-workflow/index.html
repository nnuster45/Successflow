Mockup Case Study

ระบบจัดการเอกสารบัญชีอัตโนมัติ

พัฒนา workflow ที่ใช้ Excel VBA และ Macro เพื่อช่วยดึงข้อมูลจากหลายไฟล์ธนาคาร จับคู่รายการกับเอกสาร และสรุปความผิดปกติให้ทีมบัญชีตรวจสอบได้ทันที

ปัญหา

ต้อง reconcile เอกสารจำนวนมากด้วยมือ ใช้เวลาหลายวัน

แนวทาง

รวมไฟล์อัตโนมัติ ตรวจสอบเงื่อนไข และสรุปรายการผิดปกติในหน้าเดียว

ผลลัพธ์

ลดเวลาทำงานจากหลายวันเหลือไม่กี่นาที และลด human error ในขั้นตอนสำคัญ

Project Story

โจทย์ตั้งต้นคือทีมบัญชีต้องรวมไฟล์ statement, invoice และรายการชำระเงินจากหลายแหล่งทุกวัน ก่อนตรวจสอบยอดและตามรายการผิดปกติด้วยมือ ทำให้รอบปิดงานช้าและมีความเสี่ยงเรื่อง human error สูง

Mockup Demo / GIF

วาง GIF หรือภาพ mockup flow ตรงนี้ได้ เช่น ขั้นตอน import file, match ข้อมูล, และสรุป exception โดยไม่เปิดข้อมูลลูกค้าจริง

Before / After

Before

เปิดหลายไฟล์, copy ข้อมูล, จับคู่รายการ, เช็กยอด และสรุปผิดปกติแบบ manual ใช้เวลาหลายชั่วโมงถึงหลายวันต่อรอบ

After

ผู้ใช้กดรัน workflow หนึ่งครั้ง ระบบรวมไฟล์ ตรวจสอบเงื่อนไข จับคู่ข้อมูล และสรุปรายการที่ต้องตรวจต่อให้ทันที

How It Works

01

Import ไฟล์จากหลายแหล่งเข้าโฟลเดอร์เดียว

02

Macro รวมข้อมูลและ normalize format ให้พร้อมใช้

03

ระบบ match รายการตาม rule และ flag exception

04

ทีมบัญชีเปิดสรุปผลและตรวจเฉพาะรายการที่ต้องตัดสินใจ

Confidentiality Note
