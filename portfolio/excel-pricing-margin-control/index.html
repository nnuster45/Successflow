Project Overview

Excel คำนวณราคา เสนอราคา และตรวจ Margin ก่อนอนุมัติ

โปรเจกต์นี้เหมาะกับทีมขายหรือทีมเสนอราคาที่ต้องคำนวณต้นทุน ราคาเสนอขาย ส่วนลด และ margin หลายชั้นในไฟล์เดียว โดยต้องคุม logic ให้ตรวจสอบได้ก่อนส่งอนุมัติหรือส่งลูกค้า

ปัญหา

สูตรกระจายหลายชีต คนใช้แต่ละคนแก้ไฟล์คนละแบบ ทำให้ราคาคลาดเคลื่อนและเช็ก margin ก่อนอนุมัติได้ยาก

แนวทาง

ออกแบบ workbook พร้อม VBA/Macro สำหรับคำนวณราคา ตรวจ margin และจัดชุดข้อมูลให้พร้อมส่งต่อหรือพิมพ์ใบเสนอราคา

ผลลัพธ์

ทีมเสนอราคาเร็วขึ้น ลดสูตรเพี้ยน และมี logic กลางที่ใช้ร่วมกันได้ก่อนส่งผู้อนุมัติ

Process Demonstration

วาง GIF หรือ mockup ของ pricing workbook ได้ตรงนี้ เช่น หน้า input cost, margin check, approval summary และ quotation output view

Before

ทีมต้องแก้สูตรหลายจุดเอง ใช้ไฟล์หลายเวอร์ชัน และเสียเวลาตรวจว่า margin ผ่านเกณฑ์ก่อนส่งอนุมัติหรือไม่

After

คำนวณราคาและ margin ได้ใน flow เดียว ใช้ logic กลางชุดเดียว และลดความเสี่ยงเรื่องสูตรผิดก่อนส่งต่อให้ผู้อนุมัติ

Confidentiality Note

ผลงานชิ้นนี้เป็น mockup ที่อ้างอิงจากรูปแบบการเสนอราคาจริง โดยมีการเปลี่ยนโครงสร้างข้อมูล ราคา และสูตรบางส่วนเพื่อรักษาความลับของลูกค้า
