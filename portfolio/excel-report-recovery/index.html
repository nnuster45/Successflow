Project Overview

แก้สูตรรายงานยอดขายที่พังให้กลับมาใช้ได้

กรณีศึกษาจำลองสำหรับการกู้ไฟล์รายงานยอดขายที่สูตรพัง lookup เพี้ยน และ pivot ใช้งานไม่ได้ จนทีมไม่สามารถส่งรายงานได้ตามเวลา โดยใช้ข้อมูลสมมติแทนไฟล์จริง

ปัญหา

ไฟล์หลักมีจุดพังเฉพาะหน้า ทำให้ทีมส่งรายงานยอดขายหรือปิดรอบงานไม่ได้ตามเวลา

แนวทาง

ไล่ logic, ตรวจสูตร, แก้ reference, และจัดโครงให้ไฟล์กลับมาเสถียรโดยเร็วโดยไม่ต้องรื้อใหม่ทั้งเล่ม

ผลลัพธ์

ทีมกลับมาใช้ไฟล์ต่อได้เร็วขึ้น ลดเวลาหาสาเหตุ และลดความเสี่ยงจากการแก้แบบเดาสุ่ม

Process Demonstration

วาง GIF หรือภาพ before/after ได้ตรงนี้ เช่น เซลล์ error ก่อนแก้, สูตรใหม่, pivot ที่กลับมาทำงานได้ และรายงานที่พร้อมส่งตามเวลา

Before

สูตรซ้อนกันหลายชั้น, lookup ดึงค่าผิด, pivot ไม่ refresh, และไม่มีใครแน่ใจว่าปัญหาเริ่มจากจุดไหน

After

ไฟล์กลับมาใช้งานได้, สูตรเสถียรขึ้น, logic ชัดขึ้น, และทีมสามารถส่งงานได้ทันเวลา

Confidentiality Note

ผลงานชิ้นนี้เป็น mockup ที่อ้างอิงจากปัญหา Excel จริง โดยมีการปรับโครงไฟล์ ชื่อตาราง สูตร และตัวเลขเพื่อหลีกเลี่ยงการเปิดเผยข้อมูลภายในของลูกค้า
