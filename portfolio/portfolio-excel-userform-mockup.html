Mockup Case Study

ฟอร์มบันทึกคำสั่งซื้อใน Excel แบบไม่ต้องพิมพ์ลงชีต

mockup case สำหรับระบบรับคำสั่งซื้อผ่าน Excel UserForm ที่ช่วยให้ทีมขายหรือธุรการกรอกข้อมูลได้ง่ายขึ้น ตรวจความครบถ้วนก่อนบันทึก และลดการพิมพ์ลงชีตตรง ๆ

ปัญหา

ทีมต้องพิมพ์ข้อมูลคำสั่งซื้อลง sheet ตรง ๆ ทำให้ลงผิดคอลัมน์ ข้อมูลไม่ครบ และต้องตามแก้ย้อนหลัง

แนวทาง

ออกแบบ user form ให้กรอกข้อมูลง่าย มี validation และบันทึกเข้าตารางหลักแบบเป็นระบบ

ผลลัพธ์

ข้อมูลสะอาดขึ้น, ใช้งานง่ายขึ้น, และลดปัญหาพิมพ์ผิดหรือกรอกไม่ครบก่อนส่งต่อ

Mockup Demo / GIF

วาง GIF หรือภาพ mockup ของ user form ได้ตรงนี้ เช่น หน้ากรอกข้อมูล, การ validate field, และผลลัพธ์หลังบันทึกลงตาราง

Before

ผู้ใช้พิมพ์ข้อมูลลง sheet ตรง ๆ, โอกาสลงผิดคอลัมน์สูง, และต้องตามแก้ข้อมูลย้อนหลังอยู่บ่อยครั้ง

After

ผู้ใช้กรอกผ่านฟอร์มที่อ่านง่าย, validation ชัดเจน, และข้อมูลเข้าสู่ตารางหลักอย่างเป็นระเบียบมากขึ้น

Confidentiality Note

ผลงานชิ้นนี้เป็น mockup ที่อ้างอิงจากรูปแบบการทำงานจริง โดยมีการเปลี่ยนชื่อฟอร์ม โครงข้อมูล และรายละเอียดบางส่วนเพื่อคงความลับของลูกค้า
