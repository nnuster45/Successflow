RPA / Automation

ทำ GIF 3 step: รับไฟล์เข้าโฟลเดอร์, bot รัน workflow, สรุป exception อัตโนมัติ

  • เหมาะกับ mockup หน้าจอ Excel + folder + dashboard สรุป
  • ใช้ข้อมูล dummy เช่น invoice-001, bank-a.csv
  • โชว์ before/after เวลาและจำนวนขั้นตอนที่ลดลง

Power BI Dashboard

ทำ mockup 3 screen: overview, sales trend, KPI drilldown หรือ mobile view

  • ใช้ชื่อบริษัทสมมติและตัวเลขใกล้เคียงจริงแต่ไม่ใช่ข้อมูลจริง
  • โชว์ interaction เช่น click filter แล้วกราฟเปลี่ยน
  • แนะนำใส่ callout สั้น ๆ ว่าผู้บริหารใช้อย่างไร

Excel Fast Solve

ทำ before/after ของไฟล์ที่มีสูตรพัง แล้วโชว์จุดแก้ เช่น lookup, sumifs, pivot, broken reference

  • เหมาะกับ GIF สั้น ๆ 5-10 วินาที
  • ใช้ highlight cell สีแดง -> เขียว
  • โชว์ผลลัพธ์ว่ารายงานกลับมาพร้อมส่ง

Excel UserForm

ทำ mockup form รับข้อมูล เช่น order form, expense form, stock adjustment form

  • โชว์การกรอกผ่าน form แทนการพิมพ์ลง sheet ตรง ๆ
  • ใส่ validation เช่น required field หรือ duplicate check
  • ปิดท้ายด้วยบันทึกลงตารางหรือ report อัตโนมัติ

AI Chatbot / AI Agent

ทำภาพ sequence บทสนทนา: ลูกค้าถาม, bot ตอบ, ส่งต่อทีม, บันทึก lead

  • ใช้ LINE / Facebook / Telegram mock conversation
  • โชว์ menu, FAQ, lead form, escalation ไปแอดมิน
  • ถ้ามี GIF ให้เน้น flow ไม่ต้องยาวเกิน 15 วินาที

Document / OCR Workflow

ทำ flow เอกสารเข้า -> OCR อ่านข้อมูล -> ตรวจ rule -> บันทึกลงระบบ

  • เหมาะกับภาพเปรียบเทียบก่อนและหลัง OCR
  • ทำ mockup invoice, receipt, หรือ PO ปลอม
  • โชว์จุดที่คนเข้ามาตรวจแค่ exception

Suggested Structure For Every Case

  1. Problem: ปัญหาเดิมก่อนทำ
  2. Workflow: ระบบใหม่ทำงานยังไง
  3. Mockup / GIF: ภาพประกอบหรือ animation สั้น
  4. Result: ลดเวลา ลด error ลด manual work
  5. Confidentiality Note: ระบุว่าเป็น mockup จาก use case จริงเพื่อรักษา PDPA / NDA
