Project Overview

LINE Chatbot แจ้งซ่อมและกระจายงานช่างภาคสนาม

กรณีศึกษาจำลองสำหรับงานแจ้งซ่อมผ่าน LINE ที่ต้องรับรายละเอียดหน้างาน ส่งให้ผู้ประสานงานหรือผู้อนุมัติตรวจสอบก่อน แล้วจึง assign งานให้หัวหน้าช่างหรือช่างที่รับผิดชอบในพื้นที่แบบเป็น flow เดียว

ปัญหา

งานแจ้งซ่อมเข้ามาทางแชตกระจัดกระจาย รายละเอียดไม่ครบ และต้องมีคนคอยส่งต่อ/ตามสถานะให้ช่างด้วยมือ

แนวทาง

ให้ chatbot รับข้อมูลจุดเกิดเหตุ ปัญหา รูปประกอบ และเวลานัดหมาย จากนั้นส่งต่อให้คนกลางอนุมัติและกระจายงานต่ออัตโนมัติ

ผลลัพธ์

รายละเอียดหน้างานครบขึ้น ลดงานตกหล่น และทำให้หัวหน้าช่างเห็นงานที่ต้องรับผิดชอบได้เร็วขึ้น

Field Dispatch Showcase

LINE / Field Service

แจ้งซ่อมผ่าน LINE แล้วปล่อยงานต่อให้คนอนุมัติและทีมช่างได้แบบมีลำดับ

เหมาะกับธุรกิจที่มีทีมช่างออก onsite ตามบ้าน ตามโครงการ หรือหลายพื้นที่ โดยต้องให้ข้อมูลไหลจากผู้แจ้ง > คนกลาง approve > หัวหน้าช่าง/ช่างผู้รับผิดชอบ อย่างไม่ตกหล่น

1. ลูกค้าแจ้งปัญหาผ่าน LINE พร้อมสถานที่ ประเภทงาน รูปประกอบ และช่วงเวลาที่สะดวก
2. ระบบส่งรายละเอียดให้ผู้ประสานงานหรือผู้อนุมัติตรวจสอบก่อนปล่อยงาน
3. เมื่ออนุมัติแล้ว งานถูก assign ต่อให้หัวหน้าช่างหรือช่างพื้นที่นั้น พร้อมข้อมูลครบสำหรับเข้าหน้างาน
Dispatch Simulation พื้นที่สำหรับใส่การจำลองขั้นตอน approve และ assign งานช่าง

ใช้วางภาพจำลองการแจ้งซ่อม, approve โดยคนกลาง, และการ assign ไปยังหัวหน้าช่างหรือช่าง onsite พร้อมสถานะติดตามงาน

เลื่อนไปอีกสไลด์เพื่อดูการจำลอง flow การอนุมัติและกระจายงานช่างภาคสนาม

Before

งานแจ้งซ่อมอยู่ในแชตหลายห้อง รายละเอียดหน้างานไม่ครบ และต้องมีคนตามคิวช่างกับการอนุมัติด้วยมือทุกครั้ง

After

ข้อมูลแจ้งซ่อมถูกรวบรวมเป็นระบบเดียว มีขั้นอนุมัติชัดเจน และส่งต่อให้ทีมช่างภาคสนามได้พร้อมข้อมูลครบถ้วนกว่าเดิม

Confidentiality Note

ผลงานชิ้นนี้เป็น mockup ที่อ้างอิงจาก use case งานบริการภาคสนามจริง โดยมีการเปลี่ยนรายละเอียดการแจ้งซ่อม ชื่อทีม และสถานะงานทั้งหมดเพื่อรักษาความลับของลูกค้า
