Automation

Excel Report Recovery

Project Focus

Automation

ปรับสูตร, mapping, macro และ logic ที่เกี่ยวข้องให้ไฟล์กลับมาทำงานอย่างนิ่งและตรวจสอบได้

Result

ลดเวลางมหาสาเหตุ ลดความเสี่ยงส่งรายงานผิด และช่วยให้ทีมกลับไปใช้งานไฟล์เดิมได้เร็วขึ้น
