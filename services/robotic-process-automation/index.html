Web Portal Reporting

RPA สำหรับงาน login, export, รวมไฟล์ และส่งรายงานตามรอบเวลา

เหมาะกับงานที่ทีมต้องเข้า web portal เดิมทุกวัน ดึงข้อมูลหลายรอบ รวมไฟล์ และส่งรายงานซ้ำ ๆ โดยไม่อยากเสียเวลานั่งกดหน้าจอเอง

ดูโปรเจกต์นี้

Project Focus

ดึงข้อมูลจาก web portal, export file, รวมข้อมูล และส่งรายงานตามรอบเวลา

Automation

ใช้ RPA ทำงานแทนคนบนหน้าจอเดิม โดยไม่ต้องเปลี่ยนระบบหลักขององค์กร

Result

ลดเวลางานซ้ำ ลด human error และทำให้รายงานออกได้สม่ำเสมอขึ้น

PO Mail To Planning

RPA/Automation สำหรับดึง PO จากเมล แล้วจับคู่ข้อมูลกับ DB และ ERP เพื่อทำไฟล์วางแผนการผลิต

เหมาะกับองค์กรที่มี PO จากลูกค้าเข้ามาทุกวันและต้องดึงข้อมูลหลายระบบมาประกอบกันก่อนส่งต่อให้ทีมวางแผนการผลิตหรือทีมจัดตารางงาน

ดูโปรเจกต์นี้

Project Focus

ดึงไฟล์ PO จากอีเมล แล้ว map กับข้อมูลสินค้า ลูกค้า และ ERP ก่อนสร้าง planning output

Automation

อ่านเมล, parse attachment, query SQL, match data และ transform ออกเป็น Excel สำหรับวางแผนการผลิต

Result

ช่วยให้ทีมได้ planning file เร็วขึ้น ลดงาน manual และลดความเสี่ยงจากการรวมข้อมูลผิดชุด
